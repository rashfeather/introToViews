What is Views?


...essentially a smart query builder that, given enough information, can build the proper query, execute it, and display the results. 


Source: "drupal 7" by David Mercer 

Huh?

No really, what is views?


Views is a Drupal module that allows you to:
  • Create dynamic pages
  • Based on custom selection criteria
  • Displayed how you choose
  • That is themeable
  • And exportable

When not to use views

You don't really need it

The default pages are enough
  • Blogs
  • Podcast websites
  • Simple news sites

You're customizations are minor

Write a simple module to provide your page
Or pay someone else to do it.

Views is too heavy

Views requires too much server resources for your site
  • Shared hosting
  • Very large and/or busy sites
